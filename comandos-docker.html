Comandos Básicos

  • docker --help: Entrega información sobre Docker.
  • docker info: Entrega información general sobre el sistema.
  • docker build -t <image_name>: Arma una imagen a partir de un Dockerfile
  • docker images: Lista las imágenes locales.
  • docker rmi <image_name>: Elimina una imagen.
  • docker image prune: Remueve todas las imágenes no usadas.
  • docker run --name <container_name> <image_name>: Crea y ejecuta un contenedor a partir de una imagen.
  • docker run -p <host_port>:<container_port> <image_name>: ;Ejecuta un contenedor y asigna un puerto para el host.
  • docker run -d <image_name>: Ejecuta un contenedor en segundo plano.
  • docker start <container_name>: Inicia un contenedor.
  • docker stop <container_name>: Detiene un contenedor.
  • docker rm <container_name>: Elimina un contenedor detenido.
  • docker exec -it <container_name> sh: Abre una terminal dentro de un contenedor.
  • docker ps: Lista los contenedores activos.
  • docker ps --all: Lista todos los contenedores.
